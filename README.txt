== Description ==

Just install Plugin and enjoy feature of prism Plugin.

== Installation ==
1. Upload the entire `prism-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).
